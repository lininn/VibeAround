# @va/client-ts

Shared TypeScript artifacts consumed by both the web dashboard
(`src/web/`) and the Tauri desktop UI (`src/desktop-ui/`).

## Layout

- `generated/` — types emitted from Rust via `ts-rs`. Check-in;
  regenerated by `scripts/gen-types.sh`. Do not hand-edit.
- `src/` — hand-written helpers that wrap the generated types
  (routing, URL builders, fetch wrappers). Added in Phase 3.

## Regeneration

```
scripts/gen-types.sh
```

CI will fail if the committed contents of `generated/` differ from
a fresh run.
