Feature Cards Preview
⚡

Built on ACP

All 7 agents speak the same Agent Client Protocol. Adding new agents is nearly free.

↔

Session Handover

Bidirectional handover between terminal and IM. Generate a code, pick up anywhere.

◉

Multi-Agent Multi-Channel

Claude on Telegram + Codex on Slack simultaneously. No collisions, no limits.

✦

Native IM Formatting

Feishu cards, Slack Block Kit, Discord embeds. Not a plain-text relay.
